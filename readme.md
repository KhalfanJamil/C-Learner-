Simple Calculator in C++

Description

This is a basic C++ calculator program that performs simple arithmetic operations:
	•	Addition (+)
	•	Subtraction (-)
	•	Multiplication (*)
	•	Division (/)

The program takes two numbers and an operator from the user and displays the result.

Features
	•	Easy to understand
	•	Uses switch statement
	•	Handles division by zero
	•	Uses standard input and output

How to Run
	1.	Make sure you have a C++ compiler installed (like g++).
	2.	Save the code in a file named, for example:
calculator.cpp
	3.	Compile the program:
  g++ calculator.cpp -o calculator
  ./calculator
  Enter first number: 10
Enter operator (+, -, *, /): *
Enter second number: 5
Result: 50
Code Explanation
	•	num1 and num2 store the numbers entered by the user
	•	op stores the operator
	•	switch statement selects the operation based on the operator
	•	Checks for division by zero before dividing

Requirements
	•	C++ compiler
	•	Basic knowledge of C++
